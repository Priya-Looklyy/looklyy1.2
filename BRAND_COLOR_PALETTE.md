# Looklyy Brand Color Palette

## Primary Brand Colors

### Main Purple (Primary Brand Color)
- **#7c3aed** - Violet-600
  - **Usage**: Primary buttons, icons, accents, CTAs, active states
  - **RGB**: rgb(124, 58, 237)
  - **HSL**: hsl(258, 83%, 58%)

### Secondary Purple Variants
- **#8b5cf6** - Violet-500
  - **Usage**: Loading spinners, hover states, secondary accents
  - **RGB**: rgb(139, 92, 246)
  
- **#9333ea** - Purple-600
  - **Usage**: Focus outlines, filled icons, brand highlights
  - **RGB**: rgb(147, 51, 234)

- **#a855f7** - Fuchsia-500
  - **Usage**: Gradient combinations, decorative elements
  - **RGB**: rgb(168, 85, 247)

- **#c084fc** - Purple-400
  - **Usage**: Light accents, gradient stops
  - **RGB**: rgb(192, 132, 252)

- **#6d28d9** - Violet-700
  - **Usage**: Darker purple for hover states, depth
  - **RGB**: rgb(109, 40, 217)

1. **#7c3aed → #a855f7 → #c084fc**

## Color Usage Guidelines

### Primary Brand Identity
- **Main Purple (#7c3aed)**: Use for primary CTAs, active states, brand elements
- **Purple Gradients**: Use for hero sections, premium features, backgrounds

### Text Hierarchy
- **Dark Purple (#2d1b4e)**: Primary headings and important text
- **Dark Gray (#1f2937)**: Body text and secondary headings
- **Medium Gray (#6b7280)**: Secondary text and descriptions
- **Light Gray (#9ca3af)**: Placeholders and helper text

### Interactive Elements
- **Hover States**: Darker purple (#6d28d9) or lighter purple (#8b5cf6)
- **Active States**: Main purple (#7c3aed)
- **Focus States**: Purple outline (#9333ea)

### Backgrounds
- **Main Background**: Light purple gradient (#f8f4ff → #f0ebff)
- **Card Backgrounds**: White (#ffffff) or very light purple tint
- **Section Backgrounds**: White or light gray (#f9fafb)

---

## Accessibility Notes

- **Contrast Ratios**: Ensure WCAG AA compliance (4.5:1 for normal text, 3:1 for large text)
- **Purple on White**: #7c3aed on #ffffff meets contrast requirements
- **White on Purple**: #ffffff on #7c3aed meets contrast requirements
- **Gray Text**: Use #6b7280 or darker for readable text on white backgrounds
